# ENVSYNC

**Useful CLI tool for developers**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/envsync.git
cd envsync
go build -o envsync .
```

## Usage

```bash
./envsync --help
```
